1. QUE ES RUNTU
================================================================================

Runtu es un COPILOTO DE NEGOCIO IMPULSADO POR INTELIGENCIA ARTIFICIAL, diseñado
especificamente para emprendedores y pequeños empresarios de Latinoamerica.

Es una aplicacion web que actua como asistente inteligente, permitiendo a los
usuarios comprender, analizar y mejorar sus negocios a traves de conversaciones
en lenguaje natural.

FILOSOFIA DEL PRODUCTO:
-----------------------
Runtu es "el socio que recuerda todo, entiende el caos y dice la verdad con
cariño". Esta diseñado para emprendedores que "trabajan de verdad" - no para
MBAs ni grandes corporaciones, sino para personas con:
- Una receta de la abuela
- Una habilidad aprendida a golpes
- Un local prestado
- Un WhatsApp lleno de pedidos

NOMBRE:
-------
"Runtu" probablemente deriva de "Run" + "Tu" (tu ejecutor/tu socio que corre
contigo).

2. PARA QUE SIRVE
================================================================================

FUNCIONES PRINCIPALES:
----------------------

A) CHAT INTELIGENTE CON IA
   - Responde preguntas sobre tu negocio en lenguaje natural
   - Analiza tus datos y proporciona insights accionables
   - Memoria de conversaciones previas
   - Sugerencias automaticas de seguimiento

B) PROCESAMIENTO DE DOCUMENTOS
   - Sube archivos: PDFs, Excel, CSV, imagenes, audio
   - Extraccion automatica de datos
   - Reconocimiento de texto en imagenes (OCR)
   - Transcripcion de audio
   - Organizacion inteligente de informacion

C) GENERACION DE REPORTES
   - Reportes ejecutivos (1 pagina para resumen rapido)
   - Reportes detallados (analisis completo)
   - Reportes financieros (listos para bancos/inversores)
   - Reportes operativos (operaciones del negocio)
   - Exportables a HTML, Markdown y PDF

D) ALERTAS PROACTIVAS
   - Deteccion de anomalias en tus datos
   - Alertas de tendencias importantes
   - Avisos de acciones necesarias
   - Priorizacion por urgencia (alta/media/baja)

E) RESUMENES AUTOMATICOS
   - Resumenes diarios y semanales
   - Highlights de lo mas importante
   - Generados automaticamente por IA

4. PROBLEMAS QUE SOLUCIONA
================================================================================

PROBLEMA 1: FALTA DE CLARIDAD
-----------------------------
Situacion: Los emprendedores toman decisiones "a ciegas" sin datos consolidados
Solucion: Runtu consolida toda la informacion y la presenta de forma clara

PROBLEMA 2: INFORMACION DISPERSA
--------------------------------
Situacion: Datos en WhatsApp, Excel, papeles, fotos, sin conexion entre si
Solucion: Runtu procesa y unifica todos los tipos de archivos en una sola base

PROBLEMA 3: INCAPACIDAD ANALITICA
---------------------------------
Situacion: El emprendedor no sabe como interpretar sus numeros
Solucion: Runtu analiza los datos y explica que significan en lenguaje simple

PROBLEMA 4: FALTA DE CONTEXTO HISTORICO
---------------------------------------
Situacion: Sin memoria de que paso antes, se repiten errores
Solucion: Runtu recuerda todo y puede comparar con periodos anteriores

PROBLEMA 5: DECISIONES INTUITIVAS SIN VALIDACION
------------------------------------------------
Situacion: Intuicion sin un "espejo" que la valide con datos
Solucion: Runtu valida las intuiciones del empresario con informacion real

PROBLEMA 6: SIN REPORTES PROFESIONALES
--------------------------------------
Situacion: Necesita reportes para bancos/inversores pero no sabe hacerlos
Solucion: Runtu genera reportes profesionales automaticamente

PROBLEMA 7: FALTA DE ALERTAS PROACTIVAS
---------------------------------------
Situacion: No sabe cuando algo esta mal hasta que es demasiado tarde
Solucion: Runtu detecta anomalias y alerta de forma proactiva

6. COMO FUNCIONA (FLUJO TECNICO)
================================================================================

FLUJO 1: REGISTRO Y ONBOARDING
------------------------------
1. Usuario entra a la landing page
2. Se registra con email y contraseña
3. Completa onboarding (nombre del negocio, industria)
4. Accede al dashboard principal (chat)

FLUJO 2: SUBIDA DE ARCHIVOS
---------------------------
1. Usuario sube archivo (drag & drop o click)
2. Archivo se guarda en almacenamiento seguro
3. Sistema detecta tipo de archivo automaticamente
4. Procesamiento segun tipo:
   - PDF → Extraccion de texto
   - Excel/CSV → Parseo de hojas
   - Imagen → OCR con vision artificial
   - Audio → Transcripcion automatica
5. Texto extraido se divide en fragmentos
6. Se generan embeddings (representaciones vectoriales)
7. Se guarda en base de conocimiento para consultas

FLUJO 3: CHAT CON IA
--------------------
1. Usuario escribe pregunta en lenguaje natural
2. Sistema busca informacion relevante (RAG Search)
3. Combina: pregunta + contexto + historial de conversacion
4. Gemini AI genera respuesta personalizada
5. Se muestran fuentes/documentos usados
6. Se sugieren preguntas de seguimiento

FLUJO 4: GENERACION DE REPORTES
-------------------------------
1. Usuario pide "genera reporte del mes pasado"
2. Sistema detecta intencion (tipo y periodo)
3. Extrae datos relevantes del periodo
4. IA genera reporte estructurado
5. Usuario puede exportar en multiples formatos

7. TECNOLOGIAS UTILIZADAS
================================================================================

FRONTEND:
---------
- React 19.2.0 - Componentes de interfaz
- Next.js 16.0.7 - Framework principal con SSR
- TypeScript 5 - Tipado estatico
- Tailwind CSS 4 - Estilos responsive
- Lucide React - Iconografia

BACKEND E IA:
-------------
- Google Gemini 2.0 Flash - LLM principal para chat
- Gemini 1.5 Flash Vision - Analisis de imagenes (OCR)
- Gemini 1.5 Flash Audio - Transcripcion de audio
- Anthropic Claude - LLM de respaldo

BASE DE DATOS:
--------------
- Supabase (PostgreSQL) - Base de datos principal
- Supabase Auth - Autenticacion
- Supabase Storage - Almacenamiento de archivos
- Vector Search - Busqueda semantica

PROCESAMIENTO:
--------------
- pdf-parse - Extraccion de PDFs
- SheetJS (xlsx) - Lectura de Excel/CSV
- Embeddings con Google AI - Representaciones vectoriales

SISTEMA RAG (Retrieval Augmented Generation):
---------------------------------------------
- Busqueda hibrida: palabras clave + semantica
- Chunking con overlap para contexto
- Relevancia ajustada por umbral (0.5)
- Cache de busquedas para performance

10. PROPUESTA DE VALOR UNICA
================================================================================

¿POR QUE RUNTU ES DIFERENTE?
----------------------------

1. DISEÑADO PARA LATINOAMERICA
   - Interfaz en español
   - Entiende el contexto del emprendedor latino
   - Precio accesible para la region

2. SIMPLICIDAD EXTREMA
   - Solo sube archivos y pregunta
   - Sin configuraciones complejas
   - Sin curva de aprendizaje

3. ENTIENDE EL CAOS
   - Procesa cualquier formato de archivo
   - Une datos dispersos automaticamente
   - No requiere datos "limpios"

4. PROACTIVO, NO SOLO REACTIVO
   - Genera alertas sin que le preguntes
   - Crea resumenes automaticos
   - Detecta problemas antes de que escalen

5. HABLA TU IDIOMA
   - Lenguaje simple, no tecnico
   - Respuestas claras y accionables
   - "Dice la verdad con cariño"
